Hello everyone and welcome back! In this second article\u00a0 I would like<\/em> to talk about some more ways in which Logic can make your life easier when scoring for video such as importing ideas from other project files and using stems.<\/em><\/p>\n

 <\/p>\n

Let’s assume tha<\/em>t the\u00a0<\/em>director has given me some extra notes about my cue<\/i>…happens to all of us, right?<\/em><\/p>\n

 <\/p>\n

Well it might come handy as well copy paste them in Notes<\/b> inside Logic (whose quick access you can find in the Toolbar, upper corner right). This feature isn\u00b4t exclusive of the Picture template, but it is especially helpful when you want to have quick access to your spotting notes, director\u00b4s remarks, latest stroke of inspiration…<\/p>\n

 <\/p>\n

I wish I could use some instruments and themes from the previous scene I scored<\/i><\/p>\n

 <\/p>\n

Wish granted. One of the most useful features in Logic is Import<\/b> in the Browser<\/b> of Media (ta-da!)<\/p>\n

From it you can import any input\/output source, plugin, project settings, content of track etc from any previous project into your current one. This allows you to:<\/p>\n

 <\/p>\n

    \n
  • Import that theme or motif that you used two cues ago. No need to copy-paste!<\/li>\n
  • Import those strings that you liked so much in that old project from two years ago that would go so nicely in this precise cue<\/li>\n
  • Build subtemplates of you choice (winds, strings, brass, funky synths, ethic percussion), including plugins (those lush strings with that precise reverb) that you\u00b4ll be able to coordinate flexibly as if you were working with modules<\/li>\n<\/ul>\n

    <\/a><\/p>\n

     <\/p>\n

    Stems, stems, stems…<\/i><\/p>\n

     <\/p>\n

    The stems and submixes that are often required when scoring to picture can be obtained by different procedures. Here we will see the one that should be faster!<\/p>\n

    It basically consists in assigning whatever channels of similar material you have to a particular bus, that you will later use as an independent track on its own.<\/p>\n

    Let\u00b4s say something in the line: tracks 1-7 go to bus 1 called Strings, 8-9 to bus 2 called Bass, 10 to 17 to bus 3 called Drums…you name it. By doing so you are creating some aux channels in the mixer devoted to stem groups. Now, to record those stems, we shall use the outputs of each of those submixes as input sources of new tracks in the arrange.
    \nSo, in the Arrange we create new audio tracks using the buses as input (as many as buses we created and we choose the first and go ascending). Now, if we click Record for the whole length of the track, these new tracks shall contain the stems.<\/p>\n

    That’s all ladies and gentlemen! Logic is a very powerful program and we have only touched upon very basic concepts but hopefully you have found something in here that can help you get started.<\/em><\/p>\n
